ntoskrnl-rs in the browser · ntemu

A bespoke x86-64 emulator (~60 KB WebAssembly, no threads or COOP/COEP) booting the real NT-architecture kernel: it runs both init phases, passes its self-test suite, then drops into an interactive cmd.exe. Click Boot, wait for the C:\> prompt, then type.

COM1 — ntoskrnl-rs console idle 

    

      
      
      
      
      
    

  

  
Type directly into the console once you see C:\> — try ver, echo hi, or exit.
     Enter sends the line. The kernel runs its self-tests first, so the prompt takes a few seconds.
